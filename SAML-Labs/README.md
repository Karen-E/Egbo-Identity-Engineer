## SAML Labs
Hands‑on SAML exercises demonstrating identity federation, authentication flows, metadata analysis, and troubleshooting.

### What’s in this folder
- SAML authentication flow diagram  
- Example SAML request and response  
- Metadata breakdown  
- Common SAML errors and how to fix them  
- A full SAML app integration walkthrough  

### Skills demonstrated
- Understanding SAML assertions  
- Working with IdP and SP metadata  
- Troubleshooting SSO failures  
- Mapping attributes and claims  
- Analyzing logs and error codes
