# SAML Labs
Hands-on SAML exercises demonstrating identity federation, 
authentication flows, metadata analysis, and troubleshooting.

---

## What's in this folder
- [SAML Authentication Flow diagram](SAML%20Authentication%20Flow%20diagram.png) 
  — end-to-end visual of the SAML flow from access request to 
  token validation
- [troubleshooting-scenarios.md](troubleshooting-scenarios.md) 
  — real SAML errors reproduced in a live Okta environment, 
  diagnosed with SAML Tracer, and resolved with documented 
  before/after evidence

*The following documents are in progress and will be added as 
the lab expands:*
- Example SAML request and response
- Metadata breakdown
- Full SAML app integration walkthrough

---

## Skills demonstrated
- Understanding SAML assertions
- Working with IdP and SP metadata
- Troubleshooting SSO failures using SAML Tracer
- Mapping attributes and claims
- Analyzing logs and error codes
- Reading and interpreting raw SAML XML
