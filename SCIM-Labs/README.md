- Troubleshooting SCIM sync failures

---

## Why I built this
SAML and OIDC handle authentication — proving who a user is.
SCIM handles provisioning — automatically creating, updating, and
removing user accounts across connected systems. Understanding all
three protocols gives full-spectrum coverage of how modern identity
systems manage the complete user lifecycle, not just the login
moment.
